// Planned Feature
🪙 Create Your Own Token
Launch your own token on the OTCoin network. Set your own name, symbol, and total supply — all powered by Python smart contracts.
🔬 This feature is currently in development. The API shown below represents the planned interface. Star our GitHub to get notified when it launches!
create_token.py — planned API
from smart_contract import OTCTokenContract, ContractEngine

# Initialize the contract engine
engine = ContractEngine()

# Create your token
my_token = OTCTokenContract(
    creator="YOUR_WALLET_ADDRESS",
    name="My Token",
    symbol="MTK",
    total_supply=1_000_000
)

# Deploy to OTCoin network
contract_address = engine.deploy(my_token)
print(f"Token deployed at: {contract_address}")

# Transfer tokens to another wallet
engine.call(
    contract_address, "transfer",
    {"to": "RECIPIENT_ADDRESS", "amount": 1000},
    sender="YOUR_WALLET_ADDRESS"
)
💡 Think of this like creating a new cryptocurrency that lives on top of OTCoin — just like how tokens live on Ethereum.
// Planned Feature
🎨 Launch an NFT Collection
Create and sell unique digital assets on the OTCoin blockchain. Each NFT stored permanently on-chain and can be bought, sold, and transferred.
🔬 NFT support is planned for Phase 3 of OTCoin development. The code below shows the planned API.
create_nft.py — planned API
from smart_contract import NFTContract, ContractEngine

engine = ContractEngine()

# Create NFT collection
nft = NFTContract(
    creator="YOUR_WALLET_ADDRESS",
    collection_name="My NFT Collection"
)
nft_addr = engine.deploy(nft)

# Mint NFT #1 — sell for 100 OTC
engine.call(nft_addr, "mint", {
    "metadata": {
        "name": "Genesis #1",
        "rarity": "Legendary",
        "image": "https://yoursite.com/nft1.png"
    },
    "price": 100
}, "YOUR_WALLET_ADDRESS")
💡 You can store any metadata in your NFT — name, image URL, rarity, attributes, and more!

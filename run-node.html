🚀 Quick Install (Linux/VPS)

Run this one command on your Linux server or VPS to automatically install and start an OTCoin node:

curl -sSL https://otcoin.org/node_setup.sh | bash

This script will install all dependencies, download OTCoin, and start your node automatically.

🛠️ Manual Setup
1

Install Python & Dependencies

Make sure you have Python 3.8+ installed

pip install websockets ecdsa
2

Clone OTCoin Repository

git clone https://github.com/OTCoinFoundation/OTCoin.git
cd OTCoin
3

Start Your Node

python node.py --port 6001 --peers ws://76.13.192.203:6001
4

Verify Connection

You should see your node connecting to the OTCoin network and syncing blocks.
